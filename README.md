# Synker

## Cahiers des charges

* Synchronisation automatique de fichiers via git
  * Git sert à régler les conflits éventuels qui pourraient arriver
  * Auto-commit avec date et nom du dernier paragraphe
* Éditeur web
  * Éditions (basique) et lecture de fichiers via HTML
  * Navigateur de fichiers
    * Lecture des titres Markdown à l'intérieur des fichiers
    * Recherche basique par titre, tag, contenu
  * Création d'une page web readonly pour partage
    * Possibles liens entre ces pages
* API terminal simple
  * Force sync
  * Initialiser un nouvel appareil
  * Intégrer les fonctionnalités de parcours de fichiers et de recherche de
    l'éditeur web
