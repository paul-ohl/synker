## ✍️ Typographie {#typographie}

Le markdown offre une palette d'outils typographiques élégants.

**Texte en gras** pour les idées fortes.
*Texte en italique* pour la nuance et l'emphase douce.
***Gras et italique*** quand les deux s'imposent.
~~Texte barré~~ pour ce qui est révolu.
`code inline` pour les termes techniques.

Un saut de ligne simple
crée une continuation naturelle.

Un double espace crée un nouveau paragraphe, permettant à la pensée de respirer.

## 💻 Code {#code}

### Bloc de code Python — Fibonacci élégant

```python
def fibonacci(n: int) -> list[int]:
    """Génère la séquence de Fibonacci jusqu'à n termes."""
    sequence = [0, 1]
    
    for i in range(2, n):
        sequence.append(sequence[i-1] + sequence[i-2])
    
    return sequence[:n]

# Affichage poétique
for i, num in enumerate(fibonacci(10)):
    print(f"F({i:2d}) = {num:4d}  {'█' * num if num < 50 else '█' * 50 + '...'}")
```

### Exemple JavaScript — Promesse tenue

```javascript
const fetchStargazers = async (repo) => {
  const response = await fetch(`https://api.github.com/repos/${repo}`);
  const { stargazers_count, description } = await response.json();
  
  return {
    stars: stargazers_count.toLocaleString(),
    description,
    emoji: stargazers_count > 10000 ? '🚀' : '⭐'
  };
};
```

: Langage de balisage léger utilisé pour formater du texte brut.
